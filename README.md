# Podman

<p align="center">
  <img alt="Podman" src="https://raw.githubusercontent.com/containers/common/main/logos/podman-logo-full-vert.png">
</p>

Repositório com Pods e Containers para servir de base de testes para o back-end e front-end da rádio.

## Uso

```bash
podman-compose up -d
```
ou

```bash
podman play kube pod.yml
```
